Image Processing with Python - I#

Introduce basic concepts and hands-on skills for image processing in Python using NumPy, matplotlib, and scikit-image.

Learning Outcomes:#

By the end of the session, participants will:

  • Understand images as arrays

  • Perform basic manipulations (crop, invert, rescale)

  • Apply simple filters like blur and edge detection

  • Build a reproducible image processing workflow in Python

What is a Digital Image?#

Grayscale image: 2D array, values = pixel intensity(0–255 8-bit, 0-4095 12-bit, 0-65535 16-bit)

Color image: 3D array (height, width, channels)

Libraries:#

  • numpy – numerical processing

  • matplotlib – visualization

  • skimage – image processing

Code Snippet: Load and display an image

from skimage import io
import matplotlib.pyplot as plt

image = io.imread('data/nuclei.tif')
plt.imshow(image, cmap='gray')
plt.title("Original Image")
plt.axis('off')
plt.show()

inverted = 255 - image

Rescaling Intensity#

from skimage.exposure import rescale_intensity

rescaled = rescale_intensity(image, in_range=(50, 200), out_range=(0, 255))
plt.imshow(rescaled, cmap='gray')

<matplotlib.image.AxesImage at 0x10de7e850>

Image Filtering and Enhancement#

Smoothing with Gaussian Filter#

from skimage.filters import gaussian

smooth = gaussian(image, sigma=2)
plt.imshow(smooth, cmap='gray')

<matplotlib.image.AxesImage at 0x10deaa5d0>

Edge Detection (Sobel Filter)#

from skimage.filters import sobel

edges = sobel(image)
plt.imshow(edges, cmap='gray')

<matplotlib.image.AxesImage at 0x10f804e10>

Mini Project#

Goal: Create an “image enhancer” that:

  • Loads an image from data folder

  • Crops a region of interest

  • Applies Gaussian blur and Sobel edge detection

  • Shows all steps in one figure using plt.subplot

Starter code scaffold:#

Show code cell content Hide code cell content 
fig, axes = plt.subplots(1, 4, figsize=(12, 4))
axes[0].imshow(image, cmap='gray'); axes[0].set_title('Original')
axes[1].imshow(cropped, cmap='gray'); axes[1].set_title('Cropped')
axes[2].imshow(smooth, cmap='gray'); axes[2].set_title('Blurred')
axes[3].imshow(edges, cmap='gray'); axes[3].set_title('Edges')
for ax in axes:
    ax.axis('off')
plt.tight_layout()
plt.show()
